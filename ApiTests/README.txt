# API tests for GitHub

## Description
This is a simple test automation solution which includes:
1.ApiTests class which contains api tests for GitHub
2.Configuration file to manage access variables


## Getting Started
To run the tests from this project you need to have 
- Visual Studio 
- access to GitHub api
- GitHub account, it's username and access token (to generate access token please 
visit https://docs.github.com/en/rest/authentication/authenticating-to-the-rest-api?apiVersion=2022-11-28)


### Running tests
1. Populate your username and token in appsettings.json file
2. Build the solution
3. In menu below open Test->Test Explorer
4. Click 'Run' for running all tests or right click on the test which you'd like to run
